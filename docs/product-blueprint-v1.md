# Instagram Creator Helper - Product Blueprint (V1)

## Summary
Build a fast Instagram caption assistant for small creators that turns a short post idea into hooks, captions, CTAs, and hashtag sets in one click using local Ollama models.

## Product Positioning
"Your daily Instagram writing co-pilot for small creators: idea-to-caption in under 60 seconds."

### Positioning Pillars
- Built for creators with limited time
- Structured outputs made for Instagram
- Multiple usable variations instantly
- Free-first and privacy-friendly with local AI

## Target User Persona
- Age: 18-34
- Solo creator
- Followers: 0-20k
- Posting frequency: 2-5 times/week
- Niches: fitness, travel, coding, business, lifestyle
- Main behavior: needs speed and consistency, not long strategy docs

## Exact Pain Points
- Knows what to post but not how to write it
- Caption writing slows down publishing
- Hashtag selection is repetitive and uncertain
- Needs tone variations quickly
- Wants copy-ready outputs

## Why Users Choose This Over ChatGPT
- Single-purpose workflow with no prompt engineering
- Structured output blocks for hooks/captions/CTAs/hashtags
- Multi-variation output in one run
- Niche + tone controls built in
- Local Ollama support with no recurring API cost

## MVP Feature List (Max 4)
1. Caption Generator: 3 caption variants
2. Hook Generator: 3 hook variants
3. CTA Generator: 3 CTA variants
4. Hashtag Pack Generator: low + medium + broad sets

## Monetization (Later)
- Phase A: AdSense on tool/content pages
- Phase B: Affiliate links for creator tools
- V1 focus: retention and repeat usage, no paid plan

## Differentiation Strategy
- Micro-focus on creators under 20k followers
- Speed-first output under 60 seconds
- Structured copy blocks with copy buttons
- Fast tone switching without rewriting prompt
- Zero-cost, open-source, local-AI stack

## Assumptions and Defaults
- English-first
- Text-only input in V1
- Output target: good first draft quality
- Default of 3 variations per section
